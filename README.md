# 🏠 RentWise AI Botswana

RentWise AI Botswana is an AI-powered property rental platform that helps tenants discover affordable, verified rental properties across Botswana.

## Features

- Property listings
- Property search by location
- SQLite database integration
- Flask web application
- Responsive user interface
- AI-powered rental recommendations (Coming Soon)

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Git & GitHub
